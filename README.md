# ⛄️ 燐的代購系統

這是一套**完全前端**的代購管理工具，所有資料即時取自 Google 試算表，無需自架伺服器。  
買家可直接在網頁上選購商品、計算二補運費、查詢自己的訂單狀態；團主則可透過後台查看全體訂單統計。

---

## 🚀 功能總覽

| 功能 | 說明 |
|------|------|
| 🛒 代購購物車 | 依團別選購商品、選擇款式、自動加總金額、一鍵複製訂單明細 |
| 🔎 買家查詢 | 輸入買家名稱即可查詢所有歷史訂單、匯款狀態、二補明細 |
| 🧮 二補計算機 | 自動帶入雲端運費單價，依件數計算日本國內＋國際運費 |
| 📦 團務進度表 | 顯示所有團的運回時間、狀態（待下單／已下單／已抵台） |
| 📊 後台管理 | 密碼保護，顯示全體買家訂單明細與商品總計數量 |

## 🛠️ 部署方式

### 1. 準備 Google 試算表
你需要一個已存在的試算表，內含以下**五個分頁（工作表）**，名稱必須完全一致：

| 分頁名稱 | 用途 | 格式要求 |
|----------|------|----------|
| `訂單` | 買家訂單紀錄 (gid=419241956) | 欄位：買家名、團名、狀態、購買商品、商品金額、二補 |
| `團務` | 各團進度公告 (gid=1966121219) | 欄位：團名、運回時間、狀態、二補計算說明 |
| `運費計算` | 週邊/專輯單價、日本國內運費階梯、換檔公告 | A欄：週邊品名、B欄：單價；D欄：數量上限、E欄：費用；G欄：專輯版本、H欄：國內運費、I欄：國際運費；K1/L1：公告文字 |
| `購物車` | 各團可選購的商品與價格 | 第1列：團名（A1, D1, G1... 每三欄一團）；第3列起：A欄商品名稱、B欄價格，D欄商品名稱、E欄價格... |
| `顧客填單` | 買家透過 Google 表單送出的資料 | 欄位：時間戳記、買家名稱、購買明細、總金額 |

> 試算表必須**「發佈到網路」**（檔案 → 共用 → 發佈到網路），選擇「整個文件」或個別分頁皆可。
